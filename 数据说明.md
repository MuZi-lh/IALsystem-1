#### 用户的存储的数据

- 用户可以修改该部分的数据（openid不可修改）
- openid在用户登录时可以获取，openid可以唯一标记用户

```json
{
	"openid":"...",
	"task": [
        {
            "summary": "写报告",
            "details": "写报告写报告写报告写报告写报告写报告",
            "deadline": "2020-05-30",
            "importanceLevel": 4,
            "completed": false
        }
    	],
    "item": [
        {
            "type": 0,   //0-课程  1-计划
            "name": "java",   //课程：课程名，计划：计划概述
            "remark": "江安一教A",   //课程：上课地点，计划：计划备注
            "startTime": "08:15",   //可为空字符串，注意中间冒号为英文冒号
            "endTime": "11:50",
            "repetitionType": 0,  //计划重复类型，0：不重复，1-每天重复，2-按周重复（课程是为0）
```
